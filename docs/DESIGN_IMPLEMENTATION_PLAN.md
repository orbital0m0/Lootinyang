# 🎨 Habit Cat 디자인 구현 계획

## 📋 개요

Habit Cat 앱의 디자인 시스템을 실제 UI로 구현하기 위한 상세 계획입니다. 기존 DESIGN_SYSTEM.md를 바탕으로 단계별 구현 전략을 수립합니다.

## 🎯 디자인 목표

### 핵심 목표
1. **친근하고 귀여운 캐릭터** - 고양이 캐릭터의 매력 극대화
2. **직관적인 인터페이스** - 사용하기 쉬운 UI/UX 구현
3. **게이미피케이션 요소** - 동기 부여를 위한 보상/업적 시각화
4. **모던한 미니멀리즘** - 깔끔하고 세련된 디자인

### 타겟 오디언스
- 귀여운/친근한 (40%)
- 깔끔한 모던 (30%)
- 게이미피케이션/재미 (20%)
- 직관적 사용성 (10%)

#### 표정별 디자인
1. **행복한 고양이**
   - 눈: 밝게 빛나는 눈
   - 입: 미소 짓는 입
   - 꼬리: 흔들리는 꼬리
   - 에디션: 스파클 이모지 띄우기

2. **일반 고양이**
   - 눈: 정상 눈
   - 입: 평범한 입
   - 꼬리: 가만히 있는 꼬리

3. **졸린 고양이**
   - 눈: 반쯤 감긴 눈 (ZZZ)
   - 입: 작게 벌린 입
   - 에디션: 하트 효과

4. **흥분한 고양이**
   - 눈: 커진 눈
   - 입: 떠 있는 입
   - 꼬리: 빠르게 흔들리는 꼬리

## 🎬 애니메이션 타임라인

### 페이지 로드
```css
.page-load {
  animation: fadeIn 0.3s ease-out;
}

@keyframes fadeIn {
  0% { opacity: 0; transform: translateY(20px); }
  100% { opacity: 1; transform: translateY(0); }
}
```

### 습관 체크
```css
.habit-check {
  animation: bounceIn 0.4s ease-out;
}

@keyframes bounceIn {
  0% { transform: scale(0); opacity: 0; }
  50% { transform: scale(1.2); }
  100% { transform: scale(1); opacity: 1; }
}
```

### 보상 상자 오픈
```css
.box-open {
  animation: shakeBox 0.5s ease-in-out;
}

@keyframes shakeBox {
  0%, 100% { transform: rotate(0deg); }
  25% { transform: rotate(-5deg); }
  75% { transform: rotate(5deg); }
}
```

### 레벨업
```css
.level-up {
  animation: levelUp 1s ease-out;
}

@keyframes levelUp {
  0% { transform: scale(1); filter: brightness(1); }
  50% { transform: scale(1.3); filter: brightness(1.5); }
  100% { transform: scale(1); filter: brightness(1); }
}
```

```css
  .container { max-width: 100%; }
}

/* 데스크톱 (선택적) */
@media (min-width: 1024px) {
  .container { max-width: 100%; }
}
```

## 🛠️ 기술적 구현 접근

### Framer Motion 활용
```typescript
// 페이지 전환
<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  exit={{ opacity: 0, y: -20 }}
  transition={{ duration: 0.3 }}
>
  {children}
</motion.div>

// 습관 카드 호버
<motion.div
  whileHover={{ scale: 1.02, y: -5 }}
  whileTap={{ scale: 0.98 }}
  transition={{ duration: 0.2 }}
>
  <HabitCard habit={habit} />
</motion.div>
```

### Tailwind 설정 확장
```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          50: '#fff1f2',
          100: '#ffe4e6',

          orange: '#ff8c42',

      },

        'bounce-slow': 'bounce 2s infinite',
```
